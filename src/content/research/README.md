# Research Content

This folder contains the research page content in Markdown format.

## Files

- `de.md` - German research page content
- `ja.md` - Japanese research page content

## How to Edit

Simply edit the markdown files directly! The pages will automatically update.

### Markdown Syntax

```markdown
## Section Title
This creates a section heading

### Subsection Title
This creates a subsection heading

**Bold text** for dates/emphasis
*Italic text* for titles or foreign words

- Bullet point list item
- Another item

[Link text](https://url.com)
```

## Front Matter

Each file has metadata at the top:

```yaml
---
title: 'Page Title'
description: 'SEO description'
locale: 'de' or 'ja'
keywords: ['keyword1', 'keyword2']
---
```

Don't change these unless you know what you're doing!

## Tips

- Use **bold** for dates and emphasis
- Use *italics* for book titles, foreign words, or special terms
- External links will automatically open in new tabs
- The last section with links will automatically get a divider line
